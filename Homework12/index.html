Summary: Latitude Vs. X




This analysis was done to find if the weather changes to warmer when we get closer to equator. First an amount of 600 cities were generated randomly.

We considered the Geographic Coordinates System Latitud values that are measured relative to the equator and range from -90째 at the South Pole to +90째 at the North Pole. Longitude values are measured relative to the prime meridian.

They range from -180째 when traveling west to 180째 when traveling east. Once the latitudes and longitudes were generated, the citi names were found using the citipy api. Then we found the city weather using the OpenWeatherMap API -openweather.org api-.

In this study we considered some important factors as Temperature, Humidity, Cloudiness.

The analysis results the following trends:

Trend 1

We can observe that cities located around the 0 latitud -equator line- tend to be higher the zero temperature.

Trend 2

We can observe that most of cities tend to show humidity close to 100 no matter if located close to equator or no.

Trend 3

We can observe two group of Cloudiness: a group of cities with almost zero and a second group of cities between 80 and 100 % of cloudiness no matter how close are to the equator line.
